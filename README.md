# ship_motion_model_Dlg
# MMG Ship Maneuvering Model (C++ Implementation)
A C++ implementation of the MMG ship maneuvering motion model with a Dialog-based test interface for verification and validation.

## Overview
This project provides a **C++ implementation of the standard MMG mathematical model** for ship maneuvering motion simulation.
It is designed for **initial verification, testing, and validation** of the core MMG algorithm using a lightweight Dialog-based interface.

The model can simulate basic ship maneuvering motions, including turning, yawing, and forward motion under hydrodynamic forces, making it suitable for algorithm debugging, teaching, and preliminary marine simulation research.

## Features
- Standard MMG ship maneuvering motion model in C++
- Clear, modular, and easy-to-read code structure
- Lightweight Dialog-based user interface for testing
- Real-time computation of ship motion states
- Suitable for verifying the correctness of MMG hydrodynamic equations
- Ideal for educational use and algorithm validation

## Model Background
The **MMG model** is a widely accepted mathematical model for simulating ship maneuvering motions.
It separately considers the hydrodynamic forces from:
- Hull
- Propeller
- Rudder

This separation makes the model accurate, modular, and suitable for real-time simulation and control research.

## Implementation
- **Language**: C++
- **Model**: MMG Ship Maneuvering Model
- **Interface**: Dialog-based GUI (for simple testing and visualization)
- **Purpose**: Verify the correctness and feasibility of the MMG motion model

## Usage
1. Compile the C++ source code
2. Run the executable program
3. Use the Dialog interface to initialize ship parameters
4. Start simulation to test ship maneuvering motion
5. View motion output and verify model correctness

## Application Scenarios
- Verification of MMG ship motion model algorithms
- Teaching and research of ship maneuvering theory
- Preliminary development of marine simulation systems
- Basic ship motion simulation testing
